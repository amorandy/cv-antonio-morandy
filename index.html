Plataforma Integral Financiera y Tesorería – Dimensión S.A.

Diseño y desarrollo de sistemas críticos para entidades bancarias, incluyendo gestión de créditos web, tablero de tesorería y contabilidad integral, todo interconectado mediante APIs y procesos ETL en una plataforma escalable alojada en Azure.

Entre los módulos principales se encuentra el sistema web para control de créditos, un dashboard de tesorería que administra giros aprobados en factoring, confirming y proveedores, y un sistema contable que centraliza todos los movimientos financieros.

Inicialmente, estas aplicaciones se mantenían en VBA durante 2018, ejecutándose on‑premise. En la actualidad, fueron migradas a .NET Framework 4.8 utilizando C# y DevExpress, bajo metodologías Agile para mejorar rendimiento, mantenimiento y experiencia de usuario.

La base de datos principal emplea Microsoft SQL Server versión **15.0** (SQL Server 2019), garantizando alta disponibilidad, seguridad y compatibilidad con procesos ETL automatizados.

Tecnologías: C#, .NET Framework 4.8, DevExpress, VBA (legacy), SQL Server 2019 (v 15.0), APIs REST, ETL, Azure Cloud, Agile (scrum)
